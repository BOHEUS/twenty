Twenty supports over 30 languages, and you can help us expand this list. This guide will show you how to add a new language or improve an existing translation.

## Introduction
Twenty has localization files for each language in the `src/locales` directory in `twenty-emails`, `twenty-front` and `twenty-server` packages.
Each language has a PO (Portable Object used by GNU gettext) file with translations for all the strings used in the app.
All translations are done automatically using Crowdin, a localization management platform.
